Заголовок:
- Проверка добавления курса в список желаний на https://stepic.org.
Предварительные условия:
- Пользователь зарегистрирован и авторизован на https://stepic.org.
Шаги выполнения:
- Перейти на страницу https://stepic.org.
- Нажать на поле поиска курсов.
- Ввести название курса.
- Нажать Enter.
- Выбрать из появившегося списка курсов необходимый с заданным именем.
- Открыть страницу с выбранным курсом, нажав по нему курсором.
- Нажать кнопку "Добавить в желаемое".
- Перейти на страницу "Хочу пройти в личном кабинете" и проверить наличие добавленного курса.
Ожидаемый результат:
- Выбранный курс находится в списке "Хочу пройти".
Результат:
- Выбранный курс находится в списке "Хочу пройти".
Тестовое окружение:
- Браузер Google Chrome, Версия 122.0.6261.95.
- ОС Windows 11 Pro, x64.
Заметки:
- Для выполнения теста используются несколько заданных названий курсов из списка.
- После окончания проверки наличия курса в разделе "Хочу пройти" курс удаляется из раздела для обновления данных.
